# Backdrop

### Block

| Block    | Description                                                 |
| -------- | ----------------------------------------------------------- |
| backdrop | Creates a backdrop section that pulls all content after it. |

### Modifiers

| Modifier | Description                                                 |
| -------- | ----------------------------------------------------------- |
| stretch  | Stretches sides to compensate for global gutter variable.   |
| full     | Stretches sides to fill entire viewport.                    |
